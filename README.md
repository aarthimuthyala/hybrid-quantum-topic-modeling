# Hybrid Quantum Topic Modeling

A Hybrid Quantum–Classical Optimization project for Topic Modeling and Document Clustering using FastAPI, React, NLP, and Qiskit.

## Features

- Topic Modeling using NLP techniques
- Quantum Optimization with QAOA
- Noise-aware quantum simulation
- FastAPI backend
- React frontend
- Document clustering and visualization

## Tech Stack

- Python
- FastAPI
- React + Vite
- Qiskit
- NLTK
- Pandas
- NumPy
- Matplotlib
